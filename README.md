# AI Cover Letter Generator

An AI-powered web application that helps users create professional, personalized cover letters in seconds. Simply enter your details, provide the job description, and let AI generate a tailored cover letter that matches your skills and experience.

## Live Demo

🔗 https://aicoverlettergenerator.vercel.app

## Features

- AI-powered cover letter generation
- Professional and personalized content
- User-friendly and responsive interface
- Clean and modern UI
- Fast generation process
- Mobile-friendly design
- Easy to use
- Copy generated cover letters instantly

- HTML5
- CSS3
- JavaScript

- Vite
- Google Gemini API

## Installation

1. Clone the repository

```bash
git clone https://github.com/LaibahNaushin/AI-Cover-Letter-Generator.git
```

2. Navigate to the project folder

```bash
cd AI-Cover-Letter-Generator
```

3. Install dependencies

```bash
npm install
```

4. Add your Gemini API Key

Create a `.env` file and add:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

5. Start the development server

```bash
npm run dev
```
